# Pet Project Ideas

## Selected: Personal Budget & Expense Tracker

Privacy-first, offline-capable expense tracking app with multi-currency support, internationalization, and full keyboard accessibility.

**Tech Stack:**

-   Frontend: React
-   Client Storage: IndexedDB (transactions, large data), localStorage (settings, preferences)
-   i18n: Multi-language UI, multi-currency, locale-aware formatting
-   Backend: Not needed - fully client-side

---

## Design & User Flow Resources

**UI Inspiration:**

-   Dribbble/Behance - search "budget tracker app", "expense manager dashboard", "finance app UI"
-   Figma Community - free finance/budget app templates you can fork and modify
-   Mobbin.com - real app screenshots organized by flow (filter by "finance" category)

**Study real apps for UX flows:**

-   YNAB (You Need A Budget) - best-in-class budgeting flows
-   Wallet by BudgetBakers - great multi-currency handling
-   Mint - dashboard and categorization patterns

**Accessibility references:**

-   GOV.UK Design System - gold standard for keyboard-accessible complex forms
-   Inclusive Components by Heydon Pickering (book/blog) - practical accessible patterns
